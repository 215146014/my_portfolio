1.1 IDENTIFICATION.

The project utilizes an 8-channel relay module connected to a Raspberry Pi 3 which is programmed in Python.



COMPONENTS FUNCTIONALITIES:


Raspberry Pi 3, with its Python programming, serves as the central control unit for the smart house project. It processes voice commands and sends appropriate signals to the relay module, which in turn controls the connected devices accordingly. The relay module allows for easy integration and control of multiple appliances and lights in the house.
